# 🍺 无碍理想酒馆 - H5 探索互动项目

这是一个为 **“无碍理想”** 打造的无障碍科普 H5 探索页面。

### 🚀 项目版本
**Version:** 20260503final  
**核心目标:** 通过趣味地图探索，提升大众对无障碍设计的认知。

### 🛠️ 核心功能
*   **沉浸式地图:** 适配手机竖屏的滚动探索逻辑。
*   **科普问答:** 包含 10 个关于无障碍常识的 3 选项互动题目。
*   **剧情视频:** 顺利抵达终点后触发“进入酒馆”温馨转场动画。
*   **动态奖章:** 包含实时时间戳的防伪兑换券，用于线下酒水兑换。

### 📁 文件结构说明
*   `index.html`: 项目主程序代码。
*   `asset/`: 存放核心素材（包含地图、角色、店面图及结局视频 `video.mp4`）。
